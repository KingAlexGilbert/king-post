# Security Policy

## Supported Versions

Security fixes are provided for the latest released version of King Post.

| Version | Supported |
| --- | --- |
| Latest release | Yes |
| Older releases | No |

## Reporting a Vulnerability

Please do not report security vulnerabilities through a public GitHub issue.

Instead, use the repository's **Security** tab to submit a private vulnerability report through GitHub Security Advisories.

Useful reports may involve:

- unintended access to videos or other files
- cached video copies remaining accessible longer than expected
- unsafe file-provider or content-URI behavior
- Android intent misuse
- exposure of captions or other user data
- insecure handling of temporary files
- accidental inclusion of signing keys, credentials, or private configuration

Please include:

- the affected King Post version
- your Android version and device model
- clear reproduction steps
- the expected and actual behavior
- any relevant logs or screenshots with personal information removed

Allow reasonable time for the issue to be reviewed and fixed before publicly disclosing it.
